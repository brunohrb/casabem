Configurar o Supabase

O Supabase vai ser o banco de dados central que conecta sua Alexa e o painel HTML em tempo real.

1.1 — Criar um novo projeto

  1. Acesse supabase.com e faça login com a sua conta
  2. Clique em "New Project"
  3. Escolha um nome (ex: casa-inteligente)
  4. Crie uma senha forte para o banco (guarde ela!)
  5. Escolha a região South America (São Paulo) para melhor velocidade
  6. Clique em "Create new project" e aguarde ~1 minuto

1.2 — Rodar o SQL do banco de dados

  1. No menu lateral do Supabase, clique em "SQL Editor"
  2. Clique em "New query"
  3. Abra o arquivo supabase-schema.sql que está na sua pasta
  4. Copie todo o conteúdo e cole no editor SQL
  5. Clique em "Run" (ou pressione Ctrl+Enter)
  6. Deve aparecer a mensagem "Success. No rows returned."
✅ Pronto! Suas tabelas devices, commands_log e reminders já foram criadas com 6 dispositivos de exemplo.

1.3 — Pegar as chaves de acesso

  1. No menu lateral, clique em "Project Settings" (ícone de engrenagem)
  2. Clique em "API"
  3. Você vai ver duas coisas importantes — copie e guarde:
📋 Project URL — parece com: https://abcxyz123.supabase.co

🔑 anon public (embaixo de "Project API keys") — uma chave longa começando com eyJ... — para o index.html

🔐 service_role (CUIDADO! não exponha) — para a Alexa Skill
⚠️ Atenção: A chave service_role tem acesso total ao banco. Use só no código da Alexa Skill (que roda em servidor), nunca em código público como o HTML.

Criar a Skill no Alexa Developer Console

Aqui você vai criar a "skill" (aplicativo de voz) no sistema da Amazon. É gratuito!

2.1 — Criar conta e acessar o console

  1. Acesse developer.amazon.com
  2. Faça login com a mesma conta Amazon da sua Alexa (isso é importante!)
  3. Clique em "Alexa Developer Console"

2.2 — Criar nova Skill

  1. Clique em "Create Skill"
  2. Skill name: Minha Casa
  3. Primary locale: Portuguese (BR)
  4. Model: selecione "Custom"
  5. Hosting: selecione "Alexa-hosted (Node.js)"
  6. Clique em "Create skill"
  7. Na próxima tela escolha o template "Hello World Skill" e clique "Continue with template"
  8. Aguarde a skill ser criada (~30 segundos)
✅ Sua skill foi criada! Agora vamos configurar como ela entende os comandos de voz.

2.3 — Configurar o nome de invocação

  1. No menu esquerdo, clique em "Invocation"
  2. Em "Skill Invocation Name", coloque: minha casa
  3. Clique em "Save Model"
🎙️ Com isso, você vai acionar a skill dizendo: "Alexa, minha casa" ou "Alexa, peça para minha casa ligar a luz da sala"

2.4 — Importar o modelo de interação (comandos de voz)

  1. No menu esquerdo, clique em "JSON Editor"
  2. Abra o arquivo alexa-skill/interaction-model-pt-BR.json
  3. Copie todo o conteúdo do arquivo
  4. No console, selecione tudo no editor JSON (Ctrl+A) e cole o conteúdo copiado
  5. Clique em "Save Model"
  6. Depois clique em "Build Model" e aguarde a build concluir (~1 minuto)
✅ Modelo de linguagem configurado! Sua Alexa agora entende mais de 80 frases em português para controlar dispositivos.

Colocar o código na Alexa Skill

Agora vamos colocar o código Node.js que faz a skill funcionar de verdade — recebendo comandos de voz e atualizando o Supabase.

3.1 — Acessar o Code Editor

  1. No Alexa Developer Console, clique na aba "Code" (no topo)
  2. Você vai ver um editor de código com alguns arquivos

3.2 — Substituir o index.js

  1. No painel esquerdo do editor, clique no arquivo "index.js"
  2. Abra o arquivo alexa-skill/index.js da sua pasta
  3. Copie todo o conteúdo
  4. No editor do console, selecione tudo (Ctrl+A) e cole

3.3 — Configurar as credenciais do Supabase no código

No início do arquivo index.js, você vai ver este bloco — preencha com suas credenciais:

const CONFIG = {
  SUPABASE_URL:  'https://XXXXXXXXXXXXX.supabase.co',  // ← Cole sua URL
  SUPABASE_KEY:  'eyJhbGc...',  // ← Cole a SERVICE ROLE key (não a anon!)
};
🔐 Use a chave service_role aqui (não a anon key). Ela fica segura dentro do servidor da Alexa, não fica exposta publicamente.

3.4 — Atualizar o package.json

  1. Clique no arquivo "package.json" no painel esquerdo
  2. Abra o arquivo alexa-skill/package.json da sua pasta
  3. Copie o conteúdo e substitua

3.5 — Salvar e fazer Deploy

  1. Clique em "Save" (ou Ctrl+S)
  2. Clique em "Deploy"
  3. Aguarde o deploy concluir (aparece uma barra de progresso)
✅ Código deployado! Sua skill agora está conectada ao Supabase.

3.6 — Testar no console

  1. Clique na aba "Test" no topo
  2. Mude o dropdown de "Off" para "Development"
  3. Na caixa de texto, digite: abrir minha casa
  4. Pressione Enter — você deve receber uma resposta da skill!
  5. Tente também: ligar luz da sala

Publicar o painel HTML no GitHub Pages

O GitHub Pages vai hospedar seu painel HTML gratuitamente, acessível de qualquer dispositivo pela internet.

4.1 — Antes de publicar: configurar as credenciais

Abra o arquivo index.html e localize este bloco no final:

const SUPABASE_URL = 'https://XXXXXXXXXXXXX.supabase.co'; // ← Cole aqui
const SUPABASE_KEY = 'eyJhbGc...';  // ← Cole aqui (anon key)

Aqui use a chave anon public (não a service_role). Ela é segura para ficar no HTML.

4.2 — Criar repositório no GitHub

  1. Acesse github.com e faça login
  2. Clique no botão "+" → "New repository"
  3. Nome: casa-inteligente
  4. Deixe como Public (necessário para GitHub Pages gratuito)
  5. Clique em "Create repository"

4.3 — Fazer upload dos arquivos

  1. Na página do repositório recém-criado, clique em "uploading an existing file"
  2. Arraste o arquivo index.html para a área de upload
  3. No campo de commit, escreva: Primeiro upload - painel Casa Inteligente
  4. Clique em "Commit changes"

4.4 — Ativar o GitHub Pages

  1. Dentro do repositório, clique em "Settings" (aba superior direita)
  2. No menu esquerdo, clique em "Pages"
  3. Em "Branch", selecione main e a pasta / (root)
  4. Clique em "Save"
  5. Aguarde ~1 minuto e o link vai aparecer!
✅ Seu painel estará disponível em:
https://SEU_USUARIO.github.io/casa-inteligente

4.5 — Adicionar ao celular (opcional mas recomendado!)

  1. Abra o link no seu celular
  2. No Safari (iPhone): toque em compartilhar → "Adicionar à Tela de Início"
  3. No Chrome (Android): menu → "Adicionar à tela inicial"
  4. Vai aparecer como um app na sua tela! 📱

🐛 Problemas comuns

Painel mostra "Sem conexão"
Verifique se colou corretamente a SUPABASE_URL e SUPABASE_KEY no index.html.
Alexa não encontra a skill
Certifique-se de que fez login no console com a mesma conta Amazon da sua Alexa física. A skill em modo "Development" só funciona na conta do desenvolvedor.
Skill responde mas não atualiza o banco
Verifique se a SUPABASE_KEY no index.js da skill é a service_role (não a anon) e se a URL está correta.
